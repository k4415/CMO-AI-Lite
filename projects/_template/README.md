# CMO AI Lite 案件テンプレート

このフォルダは、新規案件作成時にコピーされる空の案件テンプレートです。

商品情報、内部LP解析キャッシュ、事実、WHO-WHAT、表現レギュレーション、バナー成果物、実行ログをローカルに保存します。

## 最初のステップ（UIで操作）

1. 前提情報 > 商品情報 で商品を確認する（案件作成時に登録済み）
2. リサーチ > 事実 で「URLから事実抽出」を実行する（商品URLの内部LP解析を経由）
3. 戦略 で「WHO-WHAT生成」→ 良い案を保存する
4. 制作 > バナー でバナー案を追加し、「画像生成」を実行する

## データの場所

- 各DB: `data/*.json`
- 商品画像・ロゴ: `assets/products/{商品ID}/`
- 生成画像: `outputs/banners/{バナーID}/`
- 実行ログ: `logs/runs.md`
